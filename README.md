# launch
> A simple process manager used to run a Node.js application in detached mode

## Installation

```fish
❯ npm install @appjumpstart/launch --save
```

## Usage

Launch the process:
```fish
❯ npx launch
```

Kill the process:
```fish
❯ npx launch --kill
```
